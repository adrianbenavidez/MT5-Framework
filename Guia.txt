/---------Instalacion Inicial---------/
1 - Instalar Anaconda
2 - Ingresar a Anaconda Promnt
3 - Crear un entorno virtual: conda create -y --name mt5python python=3.11
4 - Consultar todos los entorno virtuales: conda env list
5 - Activar el entorno virtual: conda activate mt5python
6 - Desactivar el entorno virtual: conda deactivate
7 - Instalar requirements.txt: pip install -r requirements.txt
8 - Crear un archivo en el directorio raiz del proyecto Prueba.ipynb e instalar el ipykernel

/---------Modulo: Platform Connector---------/
